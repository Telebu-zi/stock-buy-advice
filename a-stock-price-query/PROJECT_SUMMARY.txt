【项目描述】
这是一个 CatPaw AI 助手的 Skill，用于查询指定 A 股股票的最近 5 日收盘价数据。
用户可以按股票代码或名称查询，系统返回收盘价、涨跌幅及统计信息。

【核心功能】
✓ 按代码查询（如："000001"）
✓ 按名称查询（如："平安银行"）
✓ 自动计算涨跌幅百分比
✓ 提供统计信息（最高、最低、平均价格）
✓ 格式化输出（表格、JSON 格式）
✓ 错误处理和提示

【使用方式】

1. 命令行运行：
   $ cd /Users/telebup/github-project/a-stock-price-query
   $ python3 fetch_stock_price.py

2. Python 模块使用：
   >>> from fetch_stock_price import StockPriceQuery
   >>> q = StockPriceQuery()
   >>> result = q.query("000001")
   >>> print(q.format_output(result))

3. 完整示例运行：

4. 在 CatPaw 中使用：
