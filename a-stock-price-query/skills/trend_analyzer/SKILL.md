# 股票趋势分析 Skill

## 功能描述

分析指定股票的趋势方向、动量、支撑阻力位，并提供综合评分和投资建议。基于技术指标和模拟数据，评估股票的投资价值。

## 使用场景

当用户需要：
- 了解某只股票的当前趋势方向（上升/下降/震荡）
- 获取技术分析指标和评分
- 查看支撑位和阻力位参考
- 比较多只股票的趋势强弱

**关键词触发**：
- "分析...趋势"
- "...股票走势如何"
- "...技术分析"
- "比较...股票"

## 输入参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|-------|------|------|------|------|
| stock_code_or_name | String | 是 | 股票代码或股票名称 | "000001" 或 "平安银行" |
| stock_codes (可选) | List[String] | 否 | 股票代码列表（用于比较） | ["000001", "000858"] |

## 执行步骤

1. **解析输入**：识别股票代码或名称
2. **数据检索**：获取股票趋势数据（模拟）
3. **指标计算**：
   - 趋势方向判断
   - 动量评估
   - 支撑阻力位识别
   - 风险等级评估
4. **综合评分**：基于加权指标计算总分（0-100）
5. **生成报告**：创建详细的分析报告和投资建议

## 技术指标说明

- **趋势方向**：上升、下降、震荡
- **动量**：强、中等、弱
- **支撑位**：预期的价格下跌支撑水平
- **阻力位**：预期的价格上涨阻力水平
- **风险等级**：低、中、高
- **移动平均线**：MA5、MA10、MA20 状态
- **摆动指标**：RSI、MACD 信号

## 错误处理

| 错误情况 | 处理方式 |
|--------|--------|
| 股票代码/名称无效 | 返回错误提示，建议确认输入 |
| 数据不可用 | 提示该股票暂无趋势分析数据 |
| 参数缺失 | 提示需要股票代码或名称 |
