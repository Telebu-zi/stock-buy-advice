# 投资建议生成 Skill

## 功能描述

基于股票数据、趋势分析和用户风险偏好，生成个性化的投资建议。包括具体操作建议、目标价位、仓位计算、风险评估和监控计划。

## 使用场景

当用户需要：
- 获取针对特定股票的投资操作建议
- 基于个人风险偏好的定制化策略
- 了解止损止盈价位和仓位管理
- 获得长期投资监控计划

**关键词触发**：
- "...投资建议"
- "应该买入还是卖出"
- "基于我的风险偏好..."
- "仓位应该多少"

## 输入参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|-------|------|------|------|------|
| stock_code | String | 是 | 股票代码 | "000001" |
| stock_name | String | 是 | 股票名称 | "平安银行" |
| current_price | Float | 是 | 当前价格 | 15.88 |
| trend_analysis | Dict | 是 | 趋势分析结果 | {"score": 72.5, "trend": "上升"} |
| user_profile | Dict | 否 | 用户风险偏好配置 | {"risk_tolerance": "平衡型", "capital": 100000} |

## 执行步骤

1. **环境评估**：分析当前市场环境（牛市/熊市/震荡市）
2. **策略匹配**：根据用户风险偏好和市场环境选择投资策略
3. **建议生成**：基于趋势分析生成具体操作建议
4. **价位计算**：计算止损、止盈、支撑、阻力价位
5. **风险评估**：评估投资风险并制定缓解措施
6. **仓位计算**：基于资金规模和风险计算建议仓位
7. **监控计划**：制定投资后监控计划

## 投资策略类型

- **保守型**：低风险，稳健收益，最大仓位30%，止损5%，止盈10%
- **平衡型**：风险收益平衡，最大仓位50%，止损10%，止盈15%
- **积极型**：高风险，高收益，最大仓位70%，止损15%，止盈20%

## 错误处理

| 错误情况 | 处理方式 |
|--------|--------|
| 缺少必要参数 | 返回错误提示，列出缺失参数 |
| 趋势分析数据无效 | 使用默认值并提示用户 |
| 用户配置不完整 | 使用默认配置（平衡型，10万资金） |
