# A 股票最近 5 日收盘价查询 Skill

## 功能描述

查询指定 A 股票的最近 5 日的收盘价。用户输入股票代码或股票名称，系统会返回该股票过去 5 个交易日的收盘价数据及相关统计信息。

## 使用场景

当用户需要：
- 查看某只 A 股股票的最近 5 日收盘价趋势
- 快速了解股票近期的价格变化
- 对比股票在过去 5 天内的价格走势

**关键词触发**：
- "查询...最近5日收盘价"
- "...股票最近5天的价格"
- "...的5日收盘价"
- "最近5个交易日...价格"

## 输入参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|-------|------|------|------|------|
| stock_code_or_name | String | 是 | 股票代码或股票名称 | "000001" 或 "平安银行" |

## 输出格式

```json
{
  "stock_code": "000001",
  "stock_name": "平安银行",
  "currency": "CNY",
  "data": [
    {
      "date": "2026-03-09",
      "close_price": 15.88,
      "change": 0.12,
      "change_percent": 0.76
    },
    {
      "date": "2026-03-06",
      "close_price": 15.76,
      "change": -0.08,
      "change_percent": -0.51
    }
  ],
  "statistics": {
    "highest_price": 15.88,
    "lowest_price": 15.50,
    "average_price": 15.72,
    "price_range": 0.38
  }
}
```

## 执行步骤

1. **解析用户输入**：识别股票代码或名称
2. **数据获取**：调用数据接口获取最近 5 个交易日的收盘价
3. **数据处理**：
   - 计算涨跌幅（相对前一日）
   - 计算统计信息（最高、最低、平均价格）
4. **结果返回**：以表格和统计形式展示数据

## 数据说明

- **close_price**：该交易日的收盘价
- **change**：相对前一交易日的价格变化（元）
- **change_percent**：相对前一交易日的百分比变化
- **price_range**：5 日内的价格跨度（最高价 - 最低价）

## 错误处理

| 错误情况 | 处理方式 |
|--------|--------|
| 股票代码/名称无效 | 返回提示并建议用户确认输入 |
| 数据不可用 | 提示用户该股票数据暂不可用 |
| 网络错误 | 返回重试提示 |
